# ACopy 复制组件

`ACopy` 是一个很好用的复制组件，可以将显示的内容一键复制到剪切板。

## Props参数

| 参数    | 类型   | 默认值 | 描述       |
| ------- | ------ | ------ | ---------- |
| content | String | -      | 复制的内容 |

## Slot插槽

| 插槽名   | 描述                                     |
| -------- | ---------------------------------------- |
| #default | 自定义显示的内容，默认为 `props.content` |
